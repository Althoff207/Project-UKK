Aplikasi Manajemen Perpustakaan
Aplikasi web perpustakaan yang memungkinkan pengelolaan buku, pengguna, dan peminjaman dengan sistem role-based access control.

Sistem perpustakaan digital ini dirancang untuk memudahkan pengelolaan perpustakaan dengan tiga tingkat akses pengguna yang berbeda. User dapat menelusuri koleksi buku berdasarkan kategori atau dari halaman utama, sementara admin dan librarian memiliki akses penuh untuk mengelola sistem.
